Esports Observer Configuration Guide

To set up your Observer, edit the following files:

Step 1: Edit game.cfg

1. Launch the game once, then close the client.

2. Navigate to:

C:\Riot Games\League of Legends\Config

3. Open game.cfg and add:

[Spectator]
eSportsNeutralTimers=1

4. Save the file and set it as Read-Only:

  • Right-click the file, select Properties.
  • Under General, check Read-Only and click Apply.

Step 2: Edit PersistedSettings.json

1. Open PersistedSettings.json in the same folder.

2. Search for:

"name": "Input.ini",
"sections": [
{
    "name": "GameEvents",
    "settings": [

3. Add this code below:

{
    "name": "evtToggleExtendedZoom",
    "value": "[Ctrl][z]"
}

4. Save the file and set it as Read-Only:

  • Right-click the file, select Properties.
  • Under General, check Read-Only and click Apply.

Esports Timers are shown when toggled with the Objecting timers Hotkey (Default: N).

Use Ctrl + Z to toggle infinite Zoom.
